Как обрабатываем базу и значения в ней. 

Что нам нужно получить из базы. 
1) Имя магазина 
	<store><![CDATA[Тест - Сайт белья]]></store>
Для XML файла только
2) Категория. Есть на польском, нужен ID категории на русском
3) Наименование. Есть на польском языке, надо на русском языке. 
4) Описание. Есть на польском языке, надо на русском языке.

5) Цвет - есть на польском, нужен код цвета и название на русском. 
6) Размер - есть на польском , нужен код размера и название на русском.
